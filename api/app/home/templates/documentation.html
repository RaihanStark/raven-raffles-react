Raven Raffles Documentation

- How to set up an account

You will get a warning if you have not set up your account. You must set up your account through the profiles and settings page. Follow the steps below to set up your account

Setup Settings

  1. Click Settings on the sidebar.
  2. Fill the Anti Captcha API (field must be filled in if you want the full features of raven raffles)
  3. Fill the Discord WebHooks URL( https://support.discordapp.com/hc/en-us/articles/228383668-Intro-to-Webhooks )
  4. Fill the Delay (in seconds) The longer the delay the less likely your proxies will be banned by them.
  5. Fill the Proxies ( One Proxies Per Line )
  6. Click Saved!

    7. Profiles Setup

To access Profiles Pages, simply click Profiles on the sidebar and fill the information.

There's two option to fill your profiles settings:

  • Fill manually on the app

  • Import from CSV file

And you're done!

- How to run the bot

The bot system here is like a cart at an online store, so you add a product to the cart (we call it task) and if you run the cart, the bot will automatically run all the tasks in the cart from top to bottom one by one.

  1. Pick the Raffles Products on Raffles Page by clicking View on your selected products
  2. Pick your size on Available Size fields (You can only pick one size per task)
  3. Input Entries Number ( it's like how much you want to submit your raffle ) for example, if you enter 3 then the bot will send 3 entries raffles
  4. Click add to task

You can repeat the number one instructions if you want to add more raffles. If you are done with that, you can follow further instructions below.

  • Click Tasks on the sidebar

There you can see what raffles the bot will send, you can delete them by clicking the trash button on the right side of the item.

  • Click Start Task

and you successfully run the bot, it is recommended not to close your browser tab when running the bot.
